# Blog Card Project

A simple and responsive blog card component built using HTML and CSS.
This project displays a blog preview card with an illustration image, publication date, title, description, and author information.

## 📌 Features

* Responsive blog card layout
* Clean and modern UI design
* Custom favicon support
* Structured semantic HTML
* Easy to customize

## 📂 Project Structure

```bash
Blog-Card/
│
├── index.html
├── style.css
│
├── assets/
│   └── images/
│       ├── favicon-32x32.png
│       ├── illustration-article.svg
│       └── image-avatar.webp
```

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/blog-card.git
```

### 2. Open the Project

Navigate to the project folder and open `index.html` in your browser.

```bash
cd blog-card
```

Then double-click `index.html`
or use Live Server in VS Code.

## 📖 Preview

The card contains:

* Article illustration image
* Category tag
* Published date
* Article title
* Short description
* Author avatar and name

## 🎨 Customization

You can easily modify:

* Colors in `style.css`
* Fonts and spacing
* Images inside `assets/images`
* Card content in `index.html`

## 📱 Responsive Design

The layout is designed to work on:

* Mobile devices
* Tablets
* Desktop screens
